Slice Navbar

The Navbar component provides a responsive navigation bar with support for logo, menu items, and action buttons. It automatically adapts to mobile devices with a hamburger menu. The component uses Static Props for automatic defaults and enhanced debugging.

With Logo

Add a logo to your navbar with the logo prop. The logo object requires a 'src' for the image path and a 'path' for navigation when clicked. Static Props ensure proper object typing and default to null when not specified.

Full Features

Combine logo, navigation items, and action buttons to create a complete navbar. The component automatically adjusts spacing based on the available elements.

Fixed Position

Set position to 'fixed' to make the navbar stay at the top of the viewport while scrolling. This is perfect for applications that need persistent navigation.

Reverse Direction

Use direction 'reverse' to flip the layout, placing the logo on the right side and menu on the left. This provides layout flexibility for different design needs.

Static Props Integration

The Navbar component leverages Static Props for improved developer experience:

  • Automatic defaults: All props have sensible defaults applied automatically
  • Type validation: Development-time warnings for incorrect prop types
  • Enhanced debugging: Right-click any navbar to see detailed prop information
  • Responsive behavior: Automatically switches to mobile menu at 1020px breakpoint
