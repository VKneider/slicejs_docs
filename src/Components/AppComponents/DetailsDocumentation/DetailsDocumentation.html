Slice Details

The Details component provides an expandable/collapsible container (accordion) perfect for showing additional information on demand. It automatically handles animation and state management. The component uses Static Props for automatic defaults and enhanced debugging.

Basic Usage

Create a simple Details component with a title and descriptive text. Click on the component to expand and collapse it. The animation is handled automatically with smooth transitions.

With Additional Content

Use the addDetail() method to add custom content inside the Details component. You can add any HTML element or Slice component, making it perfect for complex nested content.

Multiple Content Items

Add multiple elements to a single Details component to create rich, expandable sections. Great for documentation, FAQs, or any content that benefits from progressive disclosure.

Nested Details

Details components can be nested inside each other to create hierarchical expandable sections. Perfect for complex documentation or multi-level navigation.

With Code Examples

Combine Details with CodeVisualizer to create interactive code documentation. This is particularly useful for API references or tutorial content where you want to show code examples on demand.

Static Props Integration

The Details component leverages Static Props for improved developer experience:

  • Automatic defaults: All props have sensible defaults applied automatically
  • Type validation: Development-time warnings for incorrect prop types
  • Enhanced debugging: Right-click any Details to see detailed prop information
  • Self-documenting: Props are clearly defined with types and defaults
  • Smooth animations: Built-in expand/collapse animations with height transitions
