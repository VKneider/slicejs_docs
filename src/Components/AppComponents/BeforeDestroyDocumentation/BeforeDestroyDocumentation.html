beforeDestroy() Lifecycle Method

The beforeDestroy() method runs right before a component is destroyed by the Controller. Use it to clean up timers, observers, global listeners, and pending async work.

When to Use beforeDestroy()

  • Cancel intervals, timeouts, and animation frames
  • Abort pending fetch requests
  • Remove global or external event listeners
  • Dispose third-party instances (charts, maps, etc.)
