update()

Overview

update() runs whenever a component needs to refresh. This includes when cached routes are revisited via MultiRoute.

Ideal Use Cases

  • Re-fetch data that can change
  • Rebuild dynamic child components
  • Re-apply dynamic state to the DOM

Example

Best Practices

  1. Destroy old components with destroyByContainer.
  2. Clear the container.
  3. Fetch fresh data.
  4. Rebuild components with stable sliceIds.
Always destroy before recreating to avoid memory leaks.

Gotchas

Clearing innerHTML alone does not destroy Slice components.
