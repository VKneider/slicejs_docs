Slice Select

The Select component provides a customizable dropdown selector with support for single and multiple selection modes. It automatically handles option rendering and selection state. The component uses Static Props for automatic defaults and enhanced debugging.

Basic Usage

Create a simple Select with an array of options. Each option is an object where you can specify which property to display using the visibleProp. By default, it looks for a 'text' property in each option object.

With Label

Add a label (placeholder) to your Select to indicate what the user should select. The label disappears when an option is selected and reappears when cleared.

Multiple Selection

Enable multiple selection mode to allow users to select more than one option. Selected options are displayed as a comma-separated list. Static Props ensure the multiple property is properly typed as boolean with a default of false.

onOptionSelect Callback

Use the onOptionSelect callback to execute custom logic when an option is selected or deselected. This is perfect for form validation, API calls, or updating other components based on the selection.

Custom Visible Property

Use the visibleProp to specify which property of your option objects should be displayed in the dropdown. This allows you to work with complex data structures while displaying user-friendly labels.

Disabled State

Set the disabled property to true to prevent user interaction with the Select. This is useful for readonly forms or conditional form fields.

Static Props Integration

The Select component leverages Static Props for improved developer experience:

  • Automatic defaults: All props have sensible defaults applied automatically
  • Type validation: Development-time warnings for incorrect prop types
  • Enhanced debugging: Right-click any select to see detailed prop information
  • Self-documenting: Props are clearly defined with types and defaults
