Static Props Integration

The Layout component leverages Static Props for improved developer experience:

  • Automatic defaults: All props have sensible defaults applied automatically
  • Type validation: Development-time warnings for incorrect prop types
  • Enhanced debugging: Right-click any layout to see detailed prop information
  • Flexible structure: Accepts any HTMLElement or SliceJS component

Key Differences

layout prop

  • For a single fixed element
  • Set during initialization
  • Appears first in DOM order
  • Good for simple structures

onLayOut() method

  • For multiple fixed elements
  • Can be called multiple times
  • Adds elements in call order
  • More flexible and powerful
