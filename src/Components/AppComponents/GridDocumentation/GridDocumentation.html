Slice Grid

Basic Grid Layout

The Grid component provides a CSS Grid-based layout system. Simply specify the number of columns and rows, then add items using the setItem() method. Each grid uses equal-width columns (1fr) by default.

See it in action: A basic 2-column grid displaying two cards with equal spacing.

Three Column Layout

Perfect for dashboards displaying multiple metrics or data points. The Grid automatically distributes items equally across the specified columns. Great for data visualization and metric displays.

Items on Initialization

You can pass an items array when building the Grid to automatically populate it. This is perfect when you have all your content ready upfront. The Grid will add all items in order automatically.

Responsive Grid

Adapt the grid layout based on screen size by changing the columns property. This example shows 4 columns on desktop and 2 columns on mobile. You can dynamically adjust the layout based on window.innerWidth or media queries.

Static Props Integration

The Grid component leverages Static Props for improved developer experience:

  • Automatic defaults: All props have sensible defaults applied automatically
  • Type validation: Development-time warnings for incorrect prop types
  • Enhanced debugging: Right-click any grid to see detailed prop information
  • Flexible sizing: Uses CSS Grid with equal fractional units (1fr)

Common Usage Patterns

Dashboard with Metrics

const grid = await slice.build("Grid", {
   columns: 3,
   rows: 1
});

// Add metric cards
const revenueCard = await slice.build("Card", {
   title: "Revenue",
   text: "$125,000"
});
const usersCard = await slice.build("Card", {
   title: "Active Users",
   text: "1,250"
});
const salesCard = await slice.build("Card", {
   title: "Sales",
   text: "450"
});

await grid.setItem(revenueCard);
await grid.setItem(usersCard);
await grid.setItem(salesCard);

document.body.appendChild(grid);

Product Gallery

const grid = await slice.build("Grid", {
   columns: 4,
   rows: 2,
   gap: "20px"
});

// Load products dynamically
const products = await fetchProducts();
products.forEach(async (product) => {
   const productCard = await slice.build("Card", {
      title: product.name,
      text: product.price
   });
   await grid.setItem(productCard);
});

this.appendChild(grid);

Dynamic Content Management

const grid = await slice.build("Grid", {
   columns: 3,
   rows: 1
});

// Add initial items
initialItems.forEach(async (item) => {
   await grid.setItem(item);
});

// Add new item dynamically
async function addNewItem(data) {
   const card = await slice.build("Card", data);
   await grid.setItem(card);
}

// Clear and rebuild
function rebuildGrid(newItems) {
   grid.clear();
   newItems.forEach(async (item) => {
      await grid.setItem(item);
   });
}

Key Features

  • Equal Column Widths: Uses 1fr for each column (equal distribution)
  • Flexible Gap: Customizable spacing between items
  • Dynamic Addition: Add items one by one or all at once
  • Auto Grid Item Class: Each item gets grid-item class automatically
  • Responsive Ready: Easy to adapt based on screen size
  • Static Props: Automatic validation and defaults

Tips & Best Practices

  • Use the items array prop when all content is ready upfront
  • Use setItem() for dynamic content addition
  • Adjust columns based on window.innerWidth for responsive layouts
  • Use consistent gap values across your application for visual harmony
  • Call clear() before rebuilding grid content to avoid duplicates
  • Each item should have a min-height to maintain grid structure
  • Consider using rows: 1 for auto-growing grids
  • Grid items receive grid-item class for custom styling

Understanding CSS Grid

The Grid component uses native CSS Grid with repeat(n, 1fr) for columns and rows. This means each column/row takes an equal fraction of available space. The gap property controls spacing between items.

Note: The Grid component is optimized for equal-width layouts. For more complex grid patterns (like different column widths), consider using custom CSS Grid directly or extending the Grid component with columnTemplate and rowTemplate props.
