# Components Docs Portal

## Overview
Visual component API, interfaces, and examples are documented in the official
`slice.js_visual_library` docs site.

## Source of truth
- Component docs now live in one place to avoid drift and duplicate maintenance.
- `sliceDocs` keeps framework-level guidance and links to the component docs site.

## Open official components docs
Set `SLICE_PUBLIC_COMPONENTS_DOCS_URL` to your components docs deployment and open:

```bash
/Documentation/Components
```
